Sobre mim

Estudante de Desenvolvimento de Sistemas na ETEC Praia Grande, com experiência como freelancer na plataforma VintePila entre março e outubro de 2025. Desde o início, direcionei minha carreira para backend: arquitetura de APIs REST escaláveis, modelagem de dados e lógica de negócio bem estruturada, utilizando NestJS como framework principal em projetos recentes.

Busco sempre unir visão técnica e estratégica para construir sistemas que escalam.
